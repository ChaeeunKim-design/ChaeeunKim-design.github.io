# HTML의 구조

  

## HTML이란?

  

HyperText Markup Language

**웹 브라우저에 내용을 보여주는 텍스트,이미지,영상 등의 위치를 표시하는 것을 의미**

- 태그
	- \<!DOCTYPE html> : 웹 문서의 문서 유형 지정
	- \<html> : 웹 문서의 시작을 알림. 끝에는 \</html> 태그를 사용 
	- \<head> : 웹 브라우저에 문서 정보를 알려줌
	- \<title> :  문서 제목을 나타냄 
	- \<body> : 웹 브라우저에 내용을 표시
	- \<hn> : 제목
	- \<p> : text 단락을 만듦
	- \<br> : 태그 줄 바꿈
	- \<hr> : 가로줄
	- \<li> : 순서 목록을 만듦
	- \<ul> : 순서 없는 목록 만듦
	- \<dl> \<dt> \<dd> :  설명 목록 만듦 
<br /><br />
## 폼이란?

**폼(form)**

*아이디와 비밀번호를 입력하거나 로그인 버튼,회원 가입 등 사용자가 웹 사이트로 정보를 보낼 수 있는 요소*

- **\<form>** **태그**     
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;사용자가 입력한 내용을 서버 쪽 프로그램으로 어떻게 넘겨줄 것인지 지정 

- **\<fieldset>** **태그**     
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;하나의 폼 안에서 여러 구역을 나누어 표시할 때 사용 

- **\<legend>** **태그**     
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;묶은 그룹에 제목 붙임 

- **\<lable>** **태그**     
&nbsp;&nbsp;&nbsp;&nbsp;입력란 가까이에 표시하는 텍스트 

![완성본](/assets/img/label.png "label")

- **\<input>** **태그**     
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;입력하는 형태 여러 type 속성이 있음. text,password,email,radio 등
<br /><br /> 

## 로그인 폼 만들어보기 

![완성본](/assets/img/tosspage.png "tosspage")

**head**
``` html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">   
    <title>토스 앱으로 로그인</title>
    <style>
        #container {
          width:400px;
          margin:10px auto;
        }
      </style>
</head>
```

 meta -> 한글 표시하기 위해
 container -> 가로 400으로 맞춤 

**body**
``` html
<body>
    <div id="container">
        <h1>토스 앱으로 로그인</h1>
        <form>
            <fieldset>
                <ul>
                    <legend>휴대폰 번호로 로그인             QR 코드로 로그인</legend>
                    <li>
                        <label for="uname">이름</label>
                        <input type="text" id="uname" autofocus placeholder="이름" required>
                    </li>
                    <li>
                        <label for="birth">생년월일6자리</label>
                        <input type="text" id="birth" placeholder="생년월일6자리" required>
                    </li>
                    <li>
                        <lable for="phone">휴대폰 번호</lable>
                        <input type="tel" id="phone" placeholder="휴대폰 번호" required>
                    </li>

                   <br>
                   <br>
                   <li>
                    <label><input type="checkbox" name="stip" value="yes"> 필수 약관에 모두 동의</label>
                   </li>
                   <hr>
                   <li>
                    <label><input type="checkbox" name="stip" value="yes"> 개인정보 수집 및 이용 동의(토스인증서 로그인)</label>
                   </li>
                   <li>
                    <label><input type="checkbox" name="stip" value="yes"> 개인정보 제 3자 이용 동의(토스인증서 로그인)</label>
                   </li>

                   
                
                </ul>
```
 placeholder -> 힌트표시
 autofocus -> 자동 커서
 required -> 입력하지 않으면 페이지 안 넘어감
